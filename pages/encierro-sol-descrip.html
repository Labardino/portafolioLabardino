"Encierro del Sol" Project



Encierro del Sol is a 2D pixel art style videogame created in Unity.


Process description


Things got out of hand even before using the software. What we needed was a simple game mechanic which would be used in the overall game. This videogame was supposed to be done within a month so, it had to be simple but not plain. We brainstormed many ideas but ultimately we reached a verdict to the “Encierro del Sol” game mechanic. It is a puzzle game where you must find an exit
The story is that you are a bird that fell into a crack of a pyramid. You must find the exit to be reunited with your owner while avoiding the deadly traps within it (spikes and moving arrows). We decided to keep a Mexican styled theme to celebrate our country. This is why it is “Piramide del Sol” henceforth the name and Peki’s (the bird) hat.
During playtesting, many users expressed the desire of having a way to regain life, hence the coin mechanic where if you get 5 coins, your health will be restored. This project helped us to live up closely a bit the experience while developing a videogame, how many iterations of a character you go through, map design, game design, bug detection etc.

Hardships

What I loved

What I learned

One challenging part of this project was how we could easily change the arrows directions without having to do it manually, while bug detection and error was also a difficult part.
I loved the fact of how we were building a whole universe from scratch and learning more aspects of Unity. However my favorite aspect was the programming and game-design.
This project showed me a glimpse of how much work there is to do in order to build a video game. I also learned how Unity works inlcuding sprites, prefabs, coding etc.
